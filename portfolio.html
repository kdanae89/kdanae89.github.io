Connect Four

Connect Four is a classic game of alternating moves between two players, in which each player tries to make a row, column, or diagonal of four of their pieces, consecutively. I built out this online version of Connect Four using HTML5, CSS3, and JavaScript.

Photo Drop

Photo Drop is an online image hosting website. A user can register and account, log in, post an image using a url, edit their info/posts, and delete their account. I created Photo Drop using Node.js, Express, MongoDB, and Mongoose.

Project Onesie

Project Onesie allows a user to customize a onesie for a baby from ages/sizes 0-12 months. The user is given four color choices for the onesies, and a variety of images that can be dragged from the menu to the onesie. I hardcoded the images into the database and got the drag and drop functionality using JQuery UI draggable and JQuery UI droppable. Due to time constraints, I had to pivot and scale this project back a few times, but the app has user authentication going with JSON web tokens and full registration/log in functionality. Project Onesie utilizes Ruby on Rails in the backend with an AngularJS frontend. I tought myself SASS during this project and experiemented with the design. Express, Node.js, Agile development via Trello and HTML were also technologies used to create this app.
